# Versos - Poetry Collection Landing Page

A clean, beautiful landing page showcasing Spanish poetry.

## Features

- Elegant, minimal design
- Beautiful typography using Cormorant Garamond for poems
- Responsive design for all devices
- Dark mode support (system preference)
- Smooth hover animations
- Accessible and performant

## Design Principles

- Clean whitespace and breathing room
- Serif typography for poetic content
- Subtle hover effects
- Mobile-first responsive layout
- System-aware dark mode

## Tech Stack

- Pure HTML5 & CSS3
- No frameworks or dependencies
- Google Fonts (Cormorant Garamond + Inter)
- Responsive grid layout

## Deployment

Deployed on Vercel
